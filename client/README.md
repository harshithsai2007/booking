# LuxStay Frontend

The client-side application for LuxStay, a premium hotel booking platform.

## Tech Stack

- **React 18**: Modern UI library.
- **TypeScript**: Type-safe development.
- **Vite**: Ultra-fast build tool and dev server.
- **Tailwind CSS**: Utility-first styling for premium design.
- **Framer Motion**: Smooth micro-animations and transitions.
- **Axios**: API communication with JWT interceptors.

## Key Features

- **Responsive Design**: Optimized for all devices.
- **Hotel Browsing**: Seamless search and filtering experience.
- **Booking Flow**: Intuitive multi-step booking process.
- **User Dashboard**: Manage profile and view booking history.

## Development

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Build for production: `npm run build`

## Configuration

Make sure your `.env` file contains:
`VITE_API_URL=http://localhost:5000/api`
